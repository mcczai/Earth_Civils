这是EartchCivils的kubejs魔改文件夹，遵循GPL3.0协议进行开源，具体使用规则请自行查看协议。

公开是为了给在进行kubejs魔改的玩家提供一个参考，而不是进行照搬。

这个文件夹魔改涉及到：
1.机械动力、沉浸工程、农夫乐事合成修改，2.物品添加、删除
3.流体添加
4.机械动力：钻井mod的配置修改
5.tag的添加与删除
6.cgm的配方合成修改
7.物品耐久修改
8.tooltip添加以及本地化实现
9.本地化实现和其他mod模型替换
